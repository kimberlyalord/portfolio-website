Memory with Cats

"Memory with Cats" was my first major coding project created while I was at GA. The assignment was to create a browser-based game using HTML, CSS, and JavaScript. I wanted to do something I could put my personality into, and as a lifelong cat lover, I went with this spin-off of the classic game Simon, but with cat tiles instead of the original red, yellow, blue, and green buttons.

Cookie Compiler

"Cookie Compiler" is my first full-stack web app. Created with Node.js and MongoDB using the Express framework, Cookie Compiler is a recipe app that allows users to create, update, and delete their own cookie recipes, as well as read and leave reviews for any recipe. This was also my first foray into authentication (Google OAuth).

Chinchilla Collector

"Chinchilla Collector" was created using Python, Django, PostgreSQL, AWS, HTML, CSS, and Javascript. The app allows a user to create a collection of virtual chinchillas, update or delete them, add feedings and baths to a chinchilla, and add and remove toys.

GASapp

"GASapp" was a group project with fellow GA alums Travis Cashion and Chris Violante. The app was created using Python, Django, PostgreSQL, AWS, HTML, CSS, and JavaScript, and it consumes the third-party API offered by Reverb.com. GASapp is designed to be a singular, intuitive site for musicians who suffer from "Gear Acquistion Syndrome" to keep track of and valuate their music gear, and also to maintain a wishlist and connect them with links to online listings for gear they don't own yet.

Link to GitHub Repo Link to Deployed App
Learmember

"Learmember" is a portmanteau of "learn" and "remember." This full-stack app is designed to be a place where you can keep track of things that you want to learn. Users can add, update, and delete learning topics, associate learning resources with their topics, and filter lists by whether they have already learned the topic or not. Learmember was created as a single-page application using React, MongoDB, Express, Node.js, Javascript, CSS, and HTML.
